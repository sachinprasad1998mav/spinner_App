Spin Rewards

A fun little React app where users can log in and spin a reward wheel.
Each spin gives a random reward, and all your spins are shown on the rewards page.

Features
	•	Enter a 9 digit mobile number.
	•	Login with a mock OTP (123456)
	•	Animated spin-the-wheel
	•	Rewards history with timestamps
	•	Built with React, TypeScript, Tailwind, Redux Toolkit, and Vite
	•	Mock API handled by MSW (Mock Service Worker)

Run locally:
npm install
npm run dev

Open the local URL from your terminal to start the app.
Use any phone number and the OTP 123456 to log in.

Tech stack
React • TypeScript • Redux Toolkit • Tailwind • Vite • MSW • Vitest
